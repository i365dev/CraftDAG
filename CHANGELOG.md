# Changelog

## 0.2.0 - 2026-06-18

- Complete the non-redstone ComponentPlan vocabulary for architectural, landmark, landscape, section, assembly, and asset workflows.
- Add schematic import/export round trips and build artifact manifests.

- Fix `RadialRepeat` scaling, arbitrary-angle rotation, unsupported source handling, and expanded bounds diagnostics.
- Keep the public ComponentPlan and CraftDAG document schema at version `0.1`; this release changes package versions, not document compatibility.
